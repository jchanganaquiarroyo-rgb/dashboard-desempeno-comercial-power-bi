# Proyecto desarrollado como parte de mi portafolio profesional enfocado en análisis de datos y gestión de proyectos de información.

## 📌 Objetivo
Desarrollar un dashboard ejecutivo que permita analizar el desempeño comercial de la empresa, evaluando crecimiento interanual, composición de ventas y ranking estratégico de clientes y productos.

## 🛠 Tecnologías utilizadas
Power BI  
DAX  
Modelado de datos (Star Schema)  
SQL (conceptos aplicados)  
Business Intelligence
## 🧠 Enfoque

El dashboard responde tres preguntas clave:

¿Cómo estamos desempeñándonos actualmente?

¿Quiénes impulsan el negocio?

¿Cómo evolucionan nuestras ventas en el tiempo?

## 📈 Métricas Clave (DAX)

Total Ventas

Ventas Año Anterior

Variación % Interanual

Se aplicó inteligencia de tiempo usando SAMEPERIODLASTYEAR para análisis YoY.
## 📊 Componentes del Dashboard

✔ KPIs ejecutivos
✔ Análisis de composición (Treemap)
✔ Top 5 productos y clientes (Filtro Top N dinámico)
✔ Tendencia comparativa YoY
✔ Insights estratégicos escritos
